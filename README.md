# ImageConverter
Simple image to binary converter built using C++ and Qt.
There are 3 buttons on the form
1)Load image - upload any image format.png/.jpg /.jpeg

2)To binary code - if an image is uploaded, it is converted to binary code,
which is saved in the project folder

3)Into an image - selected txt file converts the binary code into a graphic image.
This image appears on the form and saving in the project folder
